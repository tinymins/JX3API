[`KScene`](./KScene.md) - Scene/Map object
[`KNpc`](./KNpc.md) - NPC object
[`KPlayer`](./KPlayer.md) - Player object
[`KDoodad`](./KDoodad.md) - Doodad (interactive object)
[`KQuestInfo`](./KQuestInfo.md) - Quest information object
