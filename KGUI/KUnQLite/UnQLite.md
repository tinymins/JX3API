## Functions

* <h4 id="LuaUnQLite_Open">UnQLite_Open</h4>
> UnQLite_Open(pcszFilePath) -> KUnQLite

Opens an UnQLite database and returns a `KUnQLite` userdata.

- `pcszFilePath`: string.

Returns nothing on failure.
